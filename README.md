# URL-Shortener
A URL-Shortener created using Node-JS and synced with MongoDB as Database.
----------------------
# Features:

* Shorten a Long URL using the web app.          
* Gets saved in MongoDB database for quick access
# Technologies involved:-


* Node JS 
* Express JS
* MongoDB
* Javascript
* HTML
* CSS (BootStrap)
